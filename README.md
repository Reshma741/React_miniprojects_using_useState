This repository contains a collection of small React projects built using the useState hook.
Each component demonstrates a different way of using state in React and can be used for learning or reference.
Projects Included
1️⃣ Day/Night Mode Toggle

A theme switcher that changes the background and text color of the page.

Features:

Switch between Day (light) and Night (dark) mode

Smooth transitions

Uses a boolean useState toggle

2️⃣ Counter App

A simple counter that increments, decrements, and resets.

Features:

+ button to increment

- button to decrement

Reset button

State fully managed using useState

3️⃣ Toggle Button

A simple toggle that switches between ON and OFF.

Features:

Boolean state toggler

Displays "ON" or "OFF"

Click to toggle

4️⃣ Update Comment

A text area that updates the comment preview in real-time.

Features:

Controlled input using useState

Live preview of typed comment

Clean and simple UI
